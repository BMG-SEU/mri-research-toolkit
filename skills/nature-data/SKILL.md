# 科研数据分析方法论 (Nature Data)

> 基于 nature-data 方法论，专为 BCI/MEG/EEG 数据分析设计

## 触发条件
用户提到「数据分析」「数据处理流程」「预处理」「分析方法选型」时加载。

## 数据驱动的科研决策框架

### Step 1：数据特征诊断
拿到数据后先回答：
1. 数据类型（连续/分段/静息态/任务态）
2. 采样率和通道数
3. 记录时长
4. 是否有事件标记（triggers/events）
5. 设备型号（影响 Maxwell 滤波等预处理选项）

### Step 2：方法选型决策树

```
数据类型？
├── 静息态 MEG
│   └→ 光谱分析 + 功能连接 + 源定位
├── 任务态 MEG/EEG
│   └→ ERP/ERF + 时频分析 + 源定位
├── EEG（高密度 64+）
│   └→ ICA + 源定位（精度低于 MEG）
└── EEG（低密度 32 以下）
    └→ 传感器层面分析为主
```

### Step 3：预处理参数速查

| 场景 | 高通 | 低通 | 陷波 | 参考 |
|------|------|------|------|------|
| ERP 分析 | 0.1 Hz | 30-40 Hz | 50/60 Hz | 平均参考 |
| 时频分析 | 1 Hz | 100 Hz | 50/60 Hz | 平均参考 |
| 源定位 | 0.1 Hz | 40 Hz | 50/60 Hz | 平均参考 |
| 功能连接 | 1 Hz | 40-100 Hz | 需注意 | 双极/平均 |

### Step 4：输出分析方案模板

```markdown
## 数据分析方案

### 数据特征
- 类型：任务态 MEG
- 采样率：1000 Hz
- 通道：306（204 梯度计 + 102 磁强计）
- 时长：每段 2s，共 300 段

### 推荐流程
1. Maxwell 滤波（降噪 + 坏通道修复）
2. 0.1-40 Hz 带通滤波
3. ICA（30 成分，剔除眼动+心跳）
4. 分段：刺激前 200ms 到刺激后 800ms
5. 基线校正：-200 到 0 ms
6. 时频分析：Morlet 小波，2-40 Hz
7. 统计：聚类置换检验，alpha=0.05，1000 次置换

### 注意事项
- 源定位前需 MRI 配准
- 功能连接需注意体积传导伪影
```
