# 论文深度阅读方法论 (Nature Reader)

> 基于 nature-reader 方法论，构建中英文对照、图表感知的论文精读笔记

## 触发条件
用户提到「读论文」「精读」「论文笔记」「paper reading」并提供 PDF/文本时加载。

## 精读框架

### Step 1：创建双语对照阅读器
对输入的论文，生成结构化的中英文对照笔记：

```markdown
# [论文标题]
> 作者 (年份) · 期刊

## Abstract
### 英文原文
...
### 中文翻译
...

## Introduction
### 英文原文（关键段落）
...
### 中文翻译 + 解读
...（本研究为什么重要）

## Methods
### 被试与实验设计
### 数据采集参数
### 预处理流程
### 分析方法

## Results
### 主要发现 1
### 主要发现 2
...

## Discussion
### 核心论点
### 局限性（作者自述的）

## 我的笔记
- 与我的研究有什么关系？
- 可以引用的点
- 方法学上可以借鉴的地方
- 需要进一步查阅的文献
```

### Step 2：图表感知
对论文中的每个图表：
1. 截图/定位图表
2. 解释图表展示的是什么
3. 提取图表中的关键数据点
4. 判断结果的可信度（效应量、样本量、误差条）

### Step 3：批判性阅读清单
- [ ] 被试数是否充分？
- [ ] 实验设计是否合理？
- [ ] 统计方法是否正确？
- [ ] 多重比较校正是否到位？
- [ ] 结论是否被数据支持？
- [ ] 有什么未讨论的局限性？

## 适用于 BCI/MEG/EEG 论文的特别关注点
- MEG 是否做了 Maxwell 滤波？
- EEG 参考方式是否合理？
- ICA 剔除成分的选择标准？
- 时频分析的小波周期数是否恰当？
- 源定位的正向模型是否说明？
