# 论文中英互译 (PaperSpine Translate)

> 基于 paper-spine-translate 方法论，逐行对照翻译，确保学术准确性

## 触发条件
用户提到「翻译」「中译英」「英译中」「translate the paper」时加载。

## 翻译流程

### Step 1：确定翻译方向
- 中→英：中文草稿 → 英文学术论文
- 英→中：英文论文 → 中文理解（课题组内部分享/组会报告）

### Step 2：逐段对照翻译

```markdown
## 原文
[原始段落]

## 译文
[翻译后段落]

## 翻译说明
- 术语处理：[列出术语的翻译选择及理由]
- 句式调整：[说明为什么调整了句式]
- 保留项：[保留的专有名词/引用]
```

### Step 3：术语一致性检查
确保通篇术语翻译一致：
- EEG → 脑电图 → 首次出现时定义
- MEG → 脑磁图
- event-related potential → 事件相关电位 (ERP)
- time-frequency → 时频
- source localization → 源定位

### Step 4：学术表达升级（中→英）
不只是翻译，而是升级为地道的英文学术表达：

| 中文草稿 | 直译（不地道） | 学术升级 |
|---------|-------------|---------|
| 我们发现...与...有关 | We found...is related to... | Our results revealed a significant association between... |
| 结果说明这个方法好用 | The results show this method works well | These findings demonstrate the effectiveness of this approach |
| 比以前的方法好 | Better than previous methods | Outperformed existing approaches (p < .01, d = 0.52) |

## 神经科学专业术语库

| 中文 | 英文 |
|------|------|
| 伪影/噪声 | artifact / noise |
| 眼电/肌电/心电 | EOG / EMG / ECG |
| 基线校正 | baseline correction |
| 诱发电位/磁场 | evoked potential / evoked field |
| 振荡 | oscillation |
| 功能连接 | functional connectivity |
| 有效连接 | effective connectivity |
| 脑网络 | brain network |
| 分类准确率 | classification accuracy |
| 信息传输率 | information transfer rate (ITR) |
