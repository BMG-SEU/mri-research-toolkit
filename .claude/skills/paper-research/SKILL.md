# 期刊调研与投稿准备 (PaperSpine Research)

> 基于 paper-spine-research 方法论，调研目标期刊要求并准备投稿材料

## 触发条件
用户提到「投哪个期刊」「选期刊」「投稿要求」「journal requirements」时加载。

## 流程

### Step 1：调研目标期刊
对候选期刊逐一分析：
```markdown
| 维度 | NeuroImage | J Neurosci Methods | IEEE TBME |
|------|-----------|-------------------|-----------|
| 影响因子 | 5.7 | 2.7 | 4.9 |
| 审稿周期 | 3-6 月 | 2-4 月 | 3-5 月 |
| 字数限制 | 6000 | 5000 | 8000 |
| 图表数量 | 无限制 | ≤8 | 无限制 |
| BCI 友好度 | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ |
| 是否需要付费 | Hybrid | 否 | 否 |
```

### Step 2：提取格式要求
```markdown
### NeuroImage 格式清单
- [ ] 摘要 ≤ 300 字
- [ ] 必须包含 Highlights (3-5 条)
- [ ] 参考文献格式：APA
- [ ] 图表：TIFF ≥ 300 DPI
- [ ] 补充材料独立文件
- [ ] 代码/数据共享声明
```

### Step 3：下载参考文献模板
提供常见 BCI 期刊的 LaTeX/Word 模板下载链接。

### Step 4：准备投稿材料清单
```markdown
## 投稿材料清单
- [ ] Manuscript（按期刊格式）
- [ ] Cover Letter
- [ ] Highlights（如需要）
- [ ] Figures（独立文件+嵌入手稿）
- [ ] Supplementary Materials
- [ ] Conflict of Interest Statement
- [ ] Author Contributions
- [ ] Data Availability Statement
```

## BCI 领域常用目标期刊
| 期刊 | 特点 |
|------|------|
| NeuroImage | BCI 神经机制研究首选 |
| J Neural Eng | BCI 工程方法首选 |
| IEEE TBME | BCI 算法和系统 |
| Frontiers in Neurosci | 开放获取，审稿快 |
| Scientific Reports | 开放获取，涵盖广 |
| Clinical Neurophysiol | 临床 BCI |
| Brain-Computer Interfaces | 专科期刊 |
